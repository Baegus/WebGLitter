# WebGLitter
Tiny WebGL 2 Particle System + Editor

## Install instructions

1. `npm install`
2. Run one of the following:

	| Command        | Description                     |
	|----------------|---------------------------------|
	| `npm run dev` | Development mode |
	| `npm run debug` | Development mode with debugging functions enabled    |
	| `npm run build` | Build the final site |

If any static assets are necessary (files that are not compiled and fetched using JS), put them into the `static` directory which acts as the root of the server.
